about

xnbox is a single file executable container that makes it possible to launch a web app as a desktop application by providing built-in server and browser functionality.
  • Tech stack: 
    		•  PWA • HTML/CSS/JavaScript • JSP/Servlets
  • App containers: 
    		•  WAR, ZIP, plain DIR
  • Operating systems: 
    		•  Linux, macOS, Windows
  • Architectures: 
    		•  x86_64, win32-x86_64, aarch64, ppc64le

    run

  • Run WAR app:
    		•     java -jar xn.jar --app MyKillerApp.war                      
  • Run static app packed as ZIP-archive:
    		•     java -jar xn.jar --app MyKillerApp.zip                      
  • Run unpacked app (dir):
    		•     java -jar xn.jar --app MyKillerApp                          
  • Run WAR (or ZIP) file from web server:
    		•     java -jar xn.jar --app https://mydomain.com/MyKillerApp.war 
    more options:  https://xnbox.github.io

    embed

    • Option 1. Copy your app content into the  /app  directory of the  xn.jar 
    • Option 2. Pack your app as  app.war  or  app.zip  and copy archive into the root directory of the  xn.jar 
    • Rename the  xn.jar  to the  MyKillerApp.jar 
    • Run your app:  java -jar --app MyKillerApp.jar 
      		•
      more options:  https://xnbox.github.io
